# IELTS Writing Assistant


A local-first UI for your IELTS agents.


## Quickstart


## 1) Clone the repo 

```
git clone https://github.com/AmirEbrahiminasab/IELTS-Assistant.git

python -m venv .venv

source .venv/bin/activate
```

# dependencies

```
pip install -r requirements.txt
```

# configure key

```
cp .env.sample .env
```

# run the app

```
python app/ui.py
```
